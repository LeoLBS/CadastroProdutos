# CadastroProdutos

##Para que serve o Cadastro de Produtos?

A sua criação foi no intuito de ajudar a fazer pequenos cadastros em um estoque de uma empresa!

##Quais as suas funções?

1. Cadastrar Produtos;
2. Pesquisar um produto no estoque;
3. Remover um cadastro inteiro;
4. Mostrar a quantidade de cadastros;
5. Produtos com o estoque zerado;
6. Adicionar e remover uma quantide de produtos do estoque;
